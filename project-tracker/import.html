新しいアーキテクチャについて

Single Source of Truth (SSoT)

GitHub mainブランチのseed_planning_data.csvが唯一の正データです。 全員が同じデータを参照します。

自動データ取得

ページを開くと、Cloudflare Worker経由で自動的にGitHubから最新データを取得します（5分キャッシュ）。

PR自動作成

新規案件登録フォームから送信すると、GitHub APIを使ってPRを自動作成します。 レビュー・マージ後、全員に反映されます。
