医師登録 — 診療科選択

担当医師の主要診療科・関連診療領域・専門分野を選択してください。

必須 の項目は登録完了に必要です。

1 主要診療科（必須・1つ選択）必須

医師の主たる専門領域を1つ選んでください。この情報は案件マッチングに使用されます。

主要診療科は必須です。

2 関連する診療領域（複数選択可）任意

主要診療科に加え、関連・隣接する診療カテゴリーをすべて選択してください。選択後に専門分野が展開されます。

3 専門分野（複数選択可）任意

上のカテゴリーを選択すると、対応する専門分野が表示されます。該当するものをすべて選んでください。

「関連する診療領域」を選択すると専門分野が表示されます

実装メモ — この画面について

  • このページは register-specialty-mock.html です。本番の register.html は変更されていません。
  • 専門科データは js/data/specialties-master.js の SPECIALTIES_MASTER オブジェクトを直接参照しています。
  • カテゴリー「血液・内分泌・代謝」「腎・泌尿器」は、マスターデータの実際のカテゴリー区分（内科系 / 泌尿器系）に合わせてマッピングしています。
  • 本番実装では、選択値を hidden input 経由で GAS に送信するか、既存の specialty フィールドとの整合性を取る必要があります。
  • UI スタイルは projects.html の .category-btn / .specialty-btn を参考に再現しています。
  • 「主要診療科」の選択肢は、カテゴリー選択と連動させる（例：選択中カテゴリーの候補のみ表示する）よう変更することも検討できます。
